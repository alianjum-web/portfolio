ABOUT

As a web developer, I have successfully worked on a variety of projects that showcase my technical proficiency, problem-solving abilities, and creative flair. These projects span different domains, demonstrating my versatility in creating dynamic and user-centric web applications. One of my most notable projects is Little Lemon, a modern restaurant website designed with both aesthetics and functionality in mind. This project highlights my expertise in creating a clean user interface (UI) that is easy to navigate and intuitive for users. I implemented responsive design to ensure the site works seamlessly across all devices and screen sizes, while also focusing on optimizing page speed for better performance. The interactive elements and user-friendly navigation make it easy for users to browse the menu, view opening hours, and place online orders.

Another key project I developed is a Spotify Clone, where I replicated many of the core features of Spotify, such as playlists, song playback, and dynamic content rendering. By working with technologies like JavaScript and React, I created a highly responsive, interactive platform that allows users to manage their playlists, stream songs, and explore new music based on their preferences. I also incorporated features like search functionality and track previews to mimic the user experience of Spotify, while integrating custom-made APIs to manage the song data and user accounts.

In addition, I worked on the Gift Project, a comprehensive web application designed to handle the entire process of buying, customizing, and sending gifts to others. The project involved building both the frontend and the backend, as well as setting up a microservices architecture to ensure scalability and flexibility in handling different services like user authentication, product recommendations, and payment processing. The backend was developed using Node.js and Express, while the frontend was powered by React and Redux for state management. The application allows users to browse gift categories, personalize products, and track deliveries, providing a smooth and secure experience from start to finish.
